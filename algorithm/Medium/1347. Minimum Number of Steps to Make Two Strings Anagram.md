# algorithm 
1347. Minimum Number of Steps to Make Two Strings Anagram

# 코드

```cpp

class Solution {
public:
    int minSteps(string s, string t) {
        map<int, int> maplist = map<int, int>();
	    for (int i = 0; i < s.size(); i++)
		    maplist[s[i]]++;

	    int count = 0;
	    for (int i = 0; i < t.size(); i++)
		    if (maplist[t[i]]-- > 0)
			    count++;
	    maplist.clear();
        return t.size() - count;
    }
};

```
