# 코드

```cpp
string solution(string number, int k) {

	string res = "";
	for (int i = 0; i < number.size(); i++)
	{		
		while (res.size() > 0 && k > 0 && res.back() < number[i])
			k--,	res.pop_back();
		res.push_back(number[i]);
	}
	while (k-- > 0)
		res.pop_back();
	return res;
}
```
