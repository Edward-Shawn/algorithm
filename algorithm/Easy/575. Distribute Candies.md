# 코드

```cpp
class Solution {
public:
int distributeCandies(vector<int>& candies) {
	return (candies.size() / 2 >= (set<int>(candies.begin(), candies.end()).size())) ? set<int>(candies.begin(), candies.end()).size() : candies.size() / 2;
}
};
```
